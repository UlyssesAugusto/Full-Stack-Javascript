Elementos de Formatação

No HTML podemos escrever utilizando negrito, itálico ou ambos para destacar partes de um texto.

As tags 'b' e 'i' são meramente para estilização e devem ser evitadas porque essa estilização deve ser feita preferencialmente com CSS. Já as tags 'strong' e 'em' devem ser utilizadas pelo seu significado, destaque e ênfase, independente dos estilos aplicados.
